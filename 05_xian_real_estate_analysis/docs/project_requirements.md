# 西安二手房房价影响因素分析与预测项目

## 1. 项目背景与目标

### 1.1 业务背景

西安作为新一线城市，近年来房地产市场发展迅速。根据国家统计局数据显示，2024年西安二手房价格同比下降约12.27%，2025年继续下行约8.77%。进入2026年，西安二手房价格在3月份成功止跌，结束了连续28个月的下跌态势，市场呈现回暖趋势。

在此背景下，深入分析影响西安二手房价格的关键因素，对于购房者和房地产从业者具有重要的参考价值。

### 1.2 分析目标

1. 分析西安各区域二手房价格分布特征
2. 识别影响二手房价格的关键因素（面积、户型、区域、楼层等）
3. 构建房价预测模型，为合理定价提供参考
4. 生成可视化报告，直观展示分析结果

### 1.3 成功指标

- 模型R²评分达到0.85以上
- 识别至少5个影响房价的关键因素
- 生成完整的数据分析报告和可视化图表

## 2. 数据资源说明

### 2.1 数据来源

本项目使用两套真实数据：

**数据源一：西安各区域二手房基础数据**
- 来源：模拟真实市场数据（基于贝壳找房、链家等平台公开数据特征）
- 说明：数据特征分布符合西安二手房市场实际情况
- 数据字段：小区名称、区域、面积、户型、楼层、总价、单价、装修程度、建造年份等

**数据源二：西安房价指数数据**
- 来源：国家统计局70个大中城市商品住宅销售价格变动情况
- 说明：官方权威数据，反映西安房价历史变化趋势
- 数据时间范围：2017年-2026年
- 数据网址：https://www.stats.gov.cn/

**数据源三：西安各区均价数据**
- 来源：58同城、安居客等公开平台
- 说明：反映西安各区域二手房均价水平
- 数据时间：2025年-2026年

### 2.2 数据规模

- 二手房基础数据：约2000条记录
- 房价指数数据：月度数据，覆盖2017-2026年
- 区域均价数据：覆盖西安11个区域

## 6. 数据字段说明

### 6.1 二手房基础数据字段

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | int | 房源唯一标识 |
| community | string | 小区名称 |
| district | string | 区域（如：雁塔区、莲湖区） |
| area | float | 面积（平方米） |
| layout | string | 户型（如：2室1厅、3室2厅） |
| floor | int | 楼层 |
| total_floors | int | 总楼层 |
| total_price | float | 总价（万元） |
| unit_price | float | 单价（元/平方米） |
| decoration | string | 装修程度（精装、毛坯、简装） |
| year_built | int | 建造年份 |
| building_type | string | 建筑类型（板楼、塔楼） |
| elevator | string | 是否有电梯（有、无） |
| orientation | string | 朝向（南、北、东、西） |

### 6.2 房价指数数据字段

| 字段名 | 类型 | 说明 |
|--------|------|------|
| month | string | 月份（YYYY-MM格式） |
| new_house_index | float | 新建商品住宅价格指数（上月=100） |
| old_house_index | float | 二手住宅价格指数（上月=100） |
| new_house_yoy | float | 新建商品住宅同比指数（上年同月=100） |
| old_house_yoy | float | 二手住宅同比指数（上年同月=100） |
