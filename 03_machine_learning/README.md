# 机器学习入门项目

这个项目介绍机器学习的基础概念，包括线性回归、分类算法和模型评估。

## 项目内容

1. **线性回归** - 预测房价示例
2. **分类算法** - 逻辑回归和 K-近邻
3. **模型评估** - 准确率、精确率、召回率、F1 分数

## 文件说明

- `data/housing_data.csv` - 示例房价数据
- `data/iris_data.csv` - 鸢尾花数据集
- `notebooks/01_linear_regression.ipynb` - 线性回归教程
- `notebooks/02_classification.ipynb` - 分类算法教程

## 数据说明

### housing_data.csv
- `area` - 房屋面积 (平方米)
- `bedrooms` - 卧室数量
- `age` - 房龄 (年)
- `price` - 房价 (万元)

### iris_data.csv
- 经典的鸢尾花数据集，用于分类任务

## 学习目标

- 理解机器学习的基本概念
- 掌握线性回归的原理和实现
- 学会使用常见的分类算法
- 能够评估模型性能
