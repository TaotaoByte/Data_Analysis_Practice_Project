# 数据可视化项目

这个项目专注于使用 Matplotlib 和 Seaborn 进行数据可视化，学习如何创建各种类型的图表来讲述数据故事。

## 项目内容

1. **基础图表** - 柱状图、折线图、散点图、饼图
2. **高级可视化** - 热力图、箱线图、小提琴图
3. **数据故事讲述** - 组合图表和自定义样式

## 文件说明

- `data/weather_data.csv` - 示例天气数据
- `notebooks/01_data_visualization.ipynb` - 数据可视化教程

## 数据说明

`weather_data.csv` 包含以下字段：
- `date` - 日期
- `city` - 城市
- `temperature` - 温度 (°C)
- `humidity` - 湿度 (%)
- `wind_speed` - 风速 (km/h)
- `precipitation` - 降水量 (mm)

## 学习目标

- 掌握 Matplotlib 的基本绘图方法
- 学会使用 Seaborn 创建美观的统计图表
- 理解不同图表类型的适用场景
- 能够创建专业的数据可视化作品
