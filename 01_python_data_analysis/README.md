# Python 数据分析入门

这是一个基础的 Python 数据分析入门项目，涵盖数据清洗、Pandas 基础操作和数据探索性分析 (EDA)。

## 项目内容

1. **数据加载和基本操作** - 使用 Pandas 加载数据
2. **数据清洗和预处理** - 处理缺失值、异常值
3. **数据探索性分析** - 统计分析和数据可视化

## 文件说明

- `data/sales_data.csv` - 示例销售数据
- `notebooks/01_data_analysis_basics.ipynb` - 数据分析基础教程

## 数据说明

`sales_data.csv` 包含以下字段：
- `date` - 销售日期
- `product` - 产品名称
- `category` - 产品类别
- `quantity` - 销售数量
- `price` - 单价
- `region` - 销售地区

## 学习目标

- 掌握 Pandas 的基本数据结构 (Series, DataFrame)
- 学会数据加载和保存
- 掌握数据清洗技巧
- 能够进行基本的数据探索和分析
